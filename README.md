### Primer proyecto de E-commerce con React

-primer proyecto con React, se realizo un e-commerce de instrumentos musicales,

-se agregaron algunos estilos con css, y otros con ChakraUi.

-se implementaron elementos tales como: Cart, Count, Form.

-se utilizo context para el contexto del carrito de compras.

-se implemento Firebase como base de datos.
